O que é a linguagem de programação C++ para ARDUÍNO

A linguagem de programação utilizada no Arduino é a linguagem C++, com pequenas modificações. Considerada uma linguagem de Alto-nível para a programação tem um grande papel quando se fala de desenvolvimento de software, ela se destaca por sua versatilidade e eficiência.
